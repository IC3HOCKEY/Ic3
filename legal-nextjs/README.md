# IC3 Legal Pages - Next.js

A Next.js application for IC3's legal pages with cookie consent.

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Run development server:
   ```
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)

## Features

- Global Header and Footer
- Legal pages: /terms, /privacy, /cookies
- Cookie consent banner with settings
- Responsive design with Montserrat and Inter fonts
- Swedish content
- SEO optimized with meta descriptions and JSON-LD schema

## Customization

- Update content in `constants/legal.js`
- Modify styles in `styles/globals.css`
- Add new pages in `pages/` directory
